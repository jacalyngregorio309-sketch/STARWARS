# Куда класть изображения

Положите файлы в папку `/assets` рядом со страницей. Имена изображений (можете потом заменить содержимое — главное сохранить имена):

- logo.png — основное лого в шапке
- stars-bg.jpg — фоновое звёздное небо (hero секция)
- hero-ship.png — главный корабль на герое (png с прозрачным фоном)
- planet-tatooine.png — планета 1 (параллакс)
- planet-hoth.png — планета 2 (параллакс)
- planet-endor.png — планета 3 (параллакс)
- about-poster.jpg — постер в блоке «О вселенной»
